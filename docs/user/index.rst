User Guide
##########

Eclipse Winery is a web-based environment to graphically model :ref:`OASIS TOSCA<tosca>` topologies and plans managing these topologies.
It is an Eclipse project and thus support is available through its `project page <https://projects.eclipse.org/projects/soa.winery>`_.
Winery is also part of the OpenTOSCA ecosystem where more information is available at `opentosca.org <http://www.opentosca.org>`_.
For more information on TOSCA see our :ref:`TOSCA information page<tosca>`.
